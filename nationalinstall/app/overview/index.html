The National Install Technology Stack

One Platform.
Every Department.
Every Roof, Tracked.

National Install runs on three connected tools — Map Measure, Eagle Eye, and the National Install App — that take a homeowner from first call to finished roof, with every photo, quote, and dollar tracked in one place.

The Three-Software Stack
📐
Measurement

Map Measure

Calculates exact roof square footage from satellite and address data — no ladder, no manual measuring tape needed before quoting.

🦅
Visualization

Eagle Eye

Overlays an AI-rendered finished roof directly onto the homeowner's actual satellite image — shown live on the Zoom call so they see their own home, transformed, before they sign.

📲
Contractor-Facing

National Install App

Entrance-to-close workflow: sq ft price lookup, step-by-step guided data entry, photo capture from all sides of the home, and client file creation — all in one guided flow.

🔄

How They Connect

A lead comes in → Secretary opens the App and creates the client file → photos upload and sync automatically → Rep pulls the address into Eagle Eye for the live overlay walkthrough → Map Measure confirms sq ft for the instant quote → everything from intake to signed deposit lives in one client file, visible to Sales, Dispatch, and Revisions.

Who Uses the Platform
DepartmentWhat They See / Do in the App
MarketingTracks campaign source per lead, cost-per-lead, and conversion by channel.
Sales (Manager, Rep, Secretary)Secretary builds the client file and uploads photos; Rep runs the Zoom quote using Eagle Eye and Map Measure; Sales Manager monitors pipeline and close rates.
RevisionsPulls the original client file to process upgrade/material change requests post-close.
DispatchReceives the completed, signed client file and hands the job to the contractor under their license — the liaison point between the sales floor and the roofer.
